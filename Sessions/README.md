# Sessions

This folder contains **raw sessions, conversations, and primary sources** used to generate insights.

## Purpose

Sessions are:
- Original dialogues or observations
- Source material for logs, archetypes, and maps
- Context for emergent patterns

## Structure

- Individual session files, e.g., `session-001-archetypal-mapping.md`
- Each session includes context, date, and relevant notes

## How to Navigate

Read sessions chronologically for full context, or search by theme or pattern to see specific insights.
